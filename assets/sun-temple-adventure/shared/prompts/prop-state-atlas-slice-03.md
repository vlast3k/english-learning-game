# Shared Prop And State Atlas Slice 03 Prompt

Purpose: create the small transparent-ready prop atlas for the Temple Steps slice.

## Image Prompt

Use case: illustration-story.
Asset type: painted point-and-click game prop atlas for chroma-key removal.

Create a small, clean painted prop and state atlas matching the existing Sun Temple Adventure style. Use a perfectly flat solid `#ff00ff` chroma-key background for later removal. The background must be one uniform color with no texture, shadow, gradient, floor, reflection, watermark, or text. Keep each prop fully separated with generous padding and crisp edges. Do not use `#ff00ff` in any prop.

Include only these ten elements, arranged in a tidy grid:

1. Empty stone cup with a small carved sun motif; simple, child-readable silhouette.
2. Water cup: the same cup visibly filled with clear blue water.
3. Opaque closed stone gate overlay, warm honey stone, front-facing and symmetrical, made to cover a fixed arch footprint.
4. Opaque open stone gate overlay with the same outer footprint, opening visible through its centre.
5. Flowing-water channel start segment, mostly straight, made for cup rest to arch.
6. Flowing-water channel gate segment, made to pass through the arch.
7. Flowing-water channel beyond-gate segment, made to continue toward the courtyard.
8. Stone Lens: warm ochre/sandstone circle with a clean sun rim and a small bright inner glow. It must read as a lens, not a coin, gem, or button.
9. Stone Lens placed-in-compass overlay.
10. Mirror Hall map marker: a locked destination emblem in the same language as a HUD map marker, never a loose world prop or collectible.

Requirements:

- no labels, letters, readable text, extra props, or decorative filler;
- no weapons, traps, skulls, keys, ropes, gems, coins, books, mirrors, or extra lenses;
- consistent soft daylight and painted texture;
- front-facing or three-quarter silhouettes suitable for small inventory icons;
- no cast shadows or contact shadows.
