# Guide Mira NPC Prompt

Purpose: create Guide Mira as a friendly jungle guide/teacher NPC for the first vertical slice.

## Image Prompt

Create a kid-friendly painted NPC character on a transparent background: Guide Mira, a warm local explorer and teacher for a jungle adventure game.

Character direction:

- friendly adult guide;
- practical explorer clothes suitable for a jungle base camp;
- warm expression, calm posture;
- no weapons;
- no franchise references;
- readable silhouette at Phaser sprite scale;
- color palette should harmonize with warm jungle camp scenes;
- should look like someone who helps children explore safely.

Pose needs:

- standing idle pose facing slightly toward the player;
- hands visible and non-threatening;
- feet visible for placement on ground;
- transparent background.

Optional:

- include 2-3 expression/head variants if feasible in the same sheet, separated clearly.

Avoid:

- photorealism;
- military styling;
- spy/thriller look;
- dense accessories that obscure silhouette.

## Integration Notes

Mira stands beside the camp table on walkable ground. Do not make her so wide that she blocks the Sun Compass or torn map click zones.
