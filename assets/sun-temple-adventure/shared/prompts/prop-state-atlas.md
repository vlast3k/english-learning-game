# Shared Prop And State Atlas Prompt

Purpose: create clean, reusable transparent-background props and state overlays for the first vertical slice.

## Image Prompt

Create a transparent-background painted prop/state atlas for a kid-friendly jungle adventure point-and-click game. Props should match a warm hand-painted adventure-game style and be readable at small inventory/icon sizes.

Include only these props and state elements:

1. Torn map piece from base camp, with clear torn edge and faint sun/river markings.
2. Smaller map piece from camp edge, with matching torn edge and river line continuation.
3. Combined valley map icon, showing joined torn edge and river line.
4. Rope coil inventory icon.
5. Rope bridge repair overlay: rope tied between simple bridge posts.
6. Lost basket with a few visible seeds inside.
7. Green Lens, bright green and gem-like but child-friendly.
8. Green Lens placed-in-compass overlay or circular green slot fill.
9. Flower pot normal state.
10. Flower pot hinted state with subtle sun mark or green shine.
11. Small drooping sun flower.
12. Small carved sun symbol motif, usable on map/stone/bridge/flower pot.
13. Waterfall Cave map marker, locked/revealed style.
14. Simple map marker icons for Base Camp, Camp Edge, Jungle Path, Broken Bridge, and Village Garden.

Requirements:

- transparent background;
- each prop separated with clear padding;
- no labels or readable text;
- no extra props;
- consistent lighting and style;
- simple silhouettes suitable for inventory and overlays;
- no weapons or scary symbols.

## Layout Recommendation

Arrange in a neat grid with enough spacing to crop each element. Keep inventory objects front-facing or three-quarter view.

## Exclusions

Do not include random tools, coins, books, skulls, weapons, extra baskets, extra ropes, or decorative filler.
