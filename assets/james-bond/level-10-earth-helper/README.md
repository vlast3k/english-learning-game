This directory is reserved for dedicated Earth Helper Mission assets.

No bitmap assets were generated for this authoring pass. The live Level 10 scenario intentionally reuses existing project assets recorded in `scenarios/drafts/james-bond-level-10-assets.json`, so no scenario path points to a missing file.

Future generated outputs should live under:

```text
assets/james-bond/level-10-earth-helper/generated/
```

When dedicated assets are added, update both `scenarios/james-bond-level-10-content.json` and `scenarios/drafts/james-bond-level-10-assets.json` together.
