# Interactive Props Prompt

Use case: illustration-story
Asset type: Clickable prop sheet for second-level hotspots
Primary request: Create a prop sheet of secret-agent surveillance apartment items for English-learning interactions.
Subjects:
- A black desk phone.
- A small camera.
- A brown agent bag.
- A compact computer monitor and keyboard.
- A room sketch/map.
- A door key tag with no readable text.
- A window clue card.
- A chair silhouette.
- A desk lamp.
- A tiny hidden microphone bug.
- A magnifying glass.
Composition: Isolated props on a light removable paper background, each object separated with generous padding for cropping into Phaser hotspot art. Arrange in a tidy grid with bold readable silhouettes at small size.
Style: Painted storybook adventure UI art matching the Level 1 secret-agent briefing props, tactile parchment/wood/metal details, safe spy theme for kids.
Lighting/mood: Warm and playful, clear edges, soft brush texture.
Constraints: No character sprites, no readable text, no logos, no watermark. Keep every object fully visible and non-overlapping.
Avoid: Guns, knives, real agency seals, real brand marks, photorealism, harsh shadows, overlapping props, tiny illegible details.
Output: Single prop sheet image.
