# Level 02: Safe Flat Watch

Kid-friendly spy adventure asset pack for the second mission in `james-bond-game.md`.

This folder follows the Level 01 asset structure but intentionally includes only the environment and item art requested for this pass. Character sprites are inherited from the current Phaser defaults until this level is fully wired with its own cast.

## Asset Files

- `generated/surveillance-apartment-background.png`: clean 1024x576 apartment surveillance room background.
- `generated/interactive-props.png`: separate prop sheet for phone, camera, bag, computer, room sketch, and related clue items.

## Prompt Files

- `prompts/surveillance-apartment-background.md`: background generation prompt.
- `prompts/interactive-props.md`: clickable prop sheet generation prompt.

## Learning Goal

Level 02 introduces `There is`, `There are`, and place prepositions:

- `in`
- `on`
- `under`
- `next to`
- `behind`

Vocabulary targets: apartment, room, door, window, phone, computer, camera, bag, desk, table, chair.
