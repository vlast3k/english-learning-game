# Interactive Props Prompt

Use case: illustration-story
Asset type: Clickable prop sheet for first-level hotspots
Primary request: Create a prop sheet of secret-agent briefing items for English-learning interactions.
Subjects:
- ID badge station with a friendly scanner light.
- Open profile folder with simple blank fields.
- Dossier board with pinned portrait cards.
- Magnifying glass.
- Stamp pad or approval stamp.
- Small table microphone.
- Mission tablet with abstract shapes, no readable text.
- Friendly gadget pouch.
Composition: Isolated props on a light removable paper background, each with generous padding for cropping into separate Phaser hotspot art. Keep shapes bold and readable at small size.
Style: Painted storybook adventure UI art matching the current Phaser campsite assets, tactile parchment and polished wood details, safe spy theme.
Avoid: Guns, knives, real agency seals, readable text, logos, photorealism, harsh shadows, overlapping props, watermarks.
Output: Single prop sheet image.
