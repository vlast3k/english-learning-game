# Agent Dossier Pictures Prompt

Use case: illustration-story
Asset type: Challenge picture set / dossier portraits
Primary request: Create a set of child-friendly dossier pictures for Level 1 identity vocabulary and `be` grammar challenges.
Scene/backdrop: Clean parchment or folder-style dossier cards viewed straight on, with simple portrait photos clipped to each card. Minimal fictional UI markings only; avoid readable agency text.
Subjects:
- A young boy agent.
- A young girl agent.
- A friendly old man mentor.
- A kind woman analyst.
- A brother and sister pair.
- A family group.
- A friend pair.
- A "good" helper character with warm expression.
- A "bad" comic-villain suspect who looks mischievous but not scary.
Vocabulary targets: name, family, friend, brother, sister, young, old, good, bad, boy, girl, man, woman.
Composition: Make each card visually distinct with clear age/gender/group cues for picture-to-word matching. Leave enough clean area around each picture for later cropping into individual challenge assets.
Style: Same painted storybook game look, warm and readable for kids, soft adventurous spy-folder framing.
Avoid: Real people, realistic IDs, official logos, frightening villains, weapons, tiny text, watermarks.
Output: One image containing all dossier pictures arranged in a tidy grid, no required readable text.
