# Level 03: Friendly Village Camp List

Kid-friendly village camp asset pack for the third mission in `james-bond-game.md`.

## Asset Files

- `generated/village-camp-background.png`: 1024x576 painted village garden camp background.
- `generated/interactive-props.png`: 1536x1024 prop sheet for bread note, football clue, teacher map, welcome board, picnic table, fruit, water, and list items.
- `generated/teacher-guide-spritesheet.png`: 768x660 teacher guide spritesheet, 4 columns by 3 rows, 192x220 frames.

## Prompt Files

- `prompts/village-camp-background.md`: background generation prompt.
- `prompts/interactive-props.md`: clickable prop sheet generation prompt.
- `prompts/teacher-guide-spritesheet.md`: teacher guide spritesheet generation prompt.

## Learning Goal

Level 03 practices simple present with `I`, `you`, `we`, and `they` in a friendly village support mission.

Vocabulary targets include village, friend, teacher, cook, garden, bread, fruit, water, football, read, help, and play.
