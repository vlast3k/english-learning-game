# Interactive Props Prompt

Create a polished painted prop atlas for children's spy adventure Level 05 Q's Lab. Match the warm hand-painted style of the existing Junior Agent Spy Academy PNG props.

Canvas: 1536x1024 bitmap prop atlas, 4 columns by 2 rows, each cell exactly 384x512.

Layout:

- Row 1, column 1: spy watch.
- Row 1, column 2: x-ray glasses with red lenses.
- Row 1, column 3: stealth jacket.
- Row 1, column 4: briefing board.
- Row 2, column 1: agent hat.
- Row 2, column 2: spy boots.
- Row 2, column 3: compact car.
- Row 2, column 4: security door.

Use a parchment-neutral background, generous padding, crisp silhouettes, warm wood/brass/navy/teal accents, and adventure-game inventory readability.

Avoid readable text, labels, logos, franchise marks, weapons, and flat vector/SVG styling.
