# Level 05: Q's Lab Gear Check

Kid-friendly laboratory asset pack for the fifth mission in `james-bond-game.md`.

## Asset Files

- `generated/q-lab-background.svg`: 1024x576 illustrated gadget laboratory background.
- `generated/interactive-props.svg`: 1536x1024, 4x2 prop atlas with 384x512 source cells.
- `generated/agent-q-spritesheet.svg`: 768x660 Agent Q guide spritesheet, 4 columns by 3 rows, 192x220 frames.

Level 05 practices `have`, `has`, `doesn't have`, questions with `Do you have...?`, possessive `'s`, and possessive adjectives through a gadget verification mission.
