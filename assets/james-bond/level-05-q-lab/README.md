# Level 05: Q's Lab Gear Check

Kid-friendly laboratory asset pack for the fifth Junior Agent Spy Academy mission.

## Asset Files

- `generated/q-lab-background.png`: 1024x576 painted gadget laboratory background.
- `generated/interactive-props.png`: 1536x1024, 4x2 prop atlas with 384x512 source cells.
- Guide sprite: reused from `../level-01-briefing/generated/director-guide-spritesheet.png`.

## Prop Atlas Layout

| Row | Column 1 | Column 2 | Column 3 | Column 4 |
| --- | --- | --- | --- | --- |
| 1 | spy watch | x-ray glasses | stealth jacket | briefing board |
| 2 | agent hat | spy boots | compact car | security door |

## Learning Goal

Level 05 practices `have`, `has`, `doesn't have`, questions with `Do you have...?`, possessive `'s`, and possessive adjectives through a gadget verification mission.
