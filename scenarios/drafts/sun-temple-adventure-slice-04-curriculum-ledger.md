# Sun Temple Adventure Slice 04 Curriculum Ledger

This ledger checks Mirror Hall language against `curriculum-scope.md`. The slice remains adventure-first: observing and turning mirrors progresses the story; there are no grammar-answer gates.

## Current Status

Status: design approved for art planning. No new player-facing Slice 4 dialogue is live yet.

## Language Policy

Use the child's visible evidence and short causal sentences:

- `The light is inside.`
- `This mirror turns the light.`
- `The light does not reach this mirror yet.`
- `The light reaches the sun mark.`
- `The panel is open.`

Avoid technical or abstract explanation in mandatory text: `reflection`, `refraction`, `angle`, `trajectory`, `optics`, `alignment`, `mechanism`, and `observatory system` are not needed for play. Use `light`, `mirror`, `line`, `turn`, `wall`, `open`, and `reach` instead.

## Grammar Coverage

| Construct | Slice-four use | Example |
| --- | --- | --- |
| `be` | visible state | `The light is inside.` `The panel is open.` |
| simple present, third-person `-s` | visible cause | `This mirror turns the light.` `The light reaches the sun mark.` |
| simple present negative | readable blocked state | `The light does not reach this mirror yet.` |
| demonstratives | direct attention | `This mirror turns the light.` |
| `can` | optional reward observation | `I can see the Mirror Lens.` |
| prepositions | optional source description | `The light is in the hall.` |

The third-person `does not` form is curriculum-covered and may appear in optional feedback. It is not presented as a grammar task. No new tense, passive construction, conditional, or science vocabulary is required for advancement.

## Coverage Decision

Mirror Hall reinforces existing concrete vocabulary and simple present cause-and-effect without exceeding the campaign's English scope. `turn`, `reach`, `panel`, and `hall` are allowed as Bulgarian-supported adventure language. The slice is ready for art-direction review and must not add grammar challenges as a substitute for the visual puzzle.
