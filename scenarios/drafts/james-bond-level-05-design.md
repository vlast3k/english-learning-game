# Level Title: Q's Lab (The Gear Check)

## Goal
The player must verify their new spy gadgets and distribute confiscated items to the correct agent profiles before leaving the lab.

## Target Grammar
- Verb `have` / `has`:
  - `I have a watch.`
  - `He has a jacket.`
  - `Do you have glasses?`
  - `She doesn't have boots.`
- Possessive `'s` and possessive adjectives:
  - `This is Q's hat.`
  - `It is my car.`
  - `Where are his boots?`
  - `Her glasses are big.`

## Target Vocabulary
- Objects: `watch`, `glasses`, `hat`, `jacket`, `boots`, `clothes`, `car`, `bike`
- Modifiers/Actions: `small`, `big`, `find`, `carry`

## Puzzle Flow
1. **Understand Mission**: Read the lab briefing board (`level05_understand_mission`).
2. **Collect Gadget Watch**: Select the spy watch, resolving grammar challenges for `have/has` (`level05_gadget_watch`).
3. **Collect X-Ray Glasses**: Select the x-ray glasses, resolving grammar questions (`level05_gadget_glasses`).
4. **Collect Stealth Jacket**: Select the stealth jacket, resolving possessive challenges (`level05_gadget_jacket`).
5. **NPC Guide (Agent Q)**: Speak with Agent Q, answering the final review check about agent gear assignments (`level05_q_approval`).
6. **Scene Exit**: Exit the lab through the security door (`level05_lab_exit`).
